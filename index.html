Game Engines

Ren'Py

Ren'Py is a visual novel engine that helps you create interactive stories using words, images, and sounds. It is open-source and easy to use, with features such as branching storylines and customizable character sprites. Ren'Py is commonly used to create dating sims and visual novels.

Supported genres Visual Novels, Dating Simulations, Life Simulations
Price Free (open source)
Commercial use allowed Yes
Where does it run Windows, macOS, Linux, iOS, Android, Browser (beta)
Requires programming skills No
Scripting languages Own, Python (optionally)

Unity

Unity is a cross-platform game engine used to develop video games for PC, consoles, mobile devices, and websites. It offers a suite of tools and services for creating 2D, 3D, AR, and VR games, as well as an integrated development environment (IDE) for writing and debugging code. Unity is popular among developers because of its flexibility and the ability to deploy games to multiple platforms with minimal effort.

Supported genres All 2D & 3D genres
Price Free and paid plans available
Commercial use allowed Yes
Where does it run Windows, macOS, Linux, iOS, Android, Browser
Requires programming skills No
Scripting languages Unity Visual Scripting, C#

Unreal Engine

Unreal Engine is a powerful game engine developed by Epic Games. It is used to create games for PC, console, mobile, and virtual reality devices. Unreal Engine features an integrated development environment (IDE) with a visual scripting system, a physics engine, and support for a wide range of graphics APIs and platforms. It is known for its cutting-edge graphics capabilities and is often used to create visually stunning games.

Supported genres All 3D genres
Price Free and paid plans available
Commercial use allowed Yes
Where does it run Windows, macOS, Linux, iOS, Android
Requires programming skills No
Scripting languages Blueprint (Visual Scripting), C++

Rpg Maker (MZ, MV, VX, XP, 2003)

RPG Maker (also known as RM) is a set of game development tools that allows users to create their own role-playing games (RPGs). It includes a visual editor, a scripting language, and a range of assets such as character sprites, backgrounds, and music. RPG Maker is available in a variety of versions, each with increasing levels of complexity and features. It is popular among game developers because of its ease of use and the ability to create games quickly.

Twine

Twine is a free, open-source tool for creating interactive fiction games, also known as "choose your own adventure" games. It allows users to create branching storylines using simple formatting and hyperlinks, and supports the use of images and sound. Twine is web-based and can be used in any modern web browser, making it easy to use and share games with others. It is often used by developers to create interactive fiction games, educational materials, and other interactive content.

Supported genres Text-based browser games
Price Free (open source)
Commercial use allowed Yes
Where does it run Browser
Requires programming skills No
Scripting languages JavaScript
